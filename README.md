# 油糕-未来金融思想实验

这是油糕（Yougao）的**未来金融思想作品原型**，非产品/工具原型——是用代码、交互承载的"对未来金融的内心感知"。

## 品牌标识
- **创作者/品牌**：油糕（Yougao）
- **作品定位**：未来金融的思想具象化表达
- **核心属性**：非产品原型，是思想作品
- **技术特点**：下一代手机银行原型，可自动适应PC WEB和APP

## 原型说明
本原型的所有页面视觉、交互、文案，均是油糕对"2034年金融该有的样子"的内心感受，不代表任何可落地的产品功能。

## 项目信息
- **原始概念**：AI Native Banking Experience
- **品牌化版本**：油糕-未来金融思想实验
- **开源协议**：MIT License
- **技术栈**：React + TypeScript + Vite + Tailwind CSS

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 访问地址
- 本地开发：http://localhost:3000

## 项目结构
```
├── src/
│   ├── App.tsx              # 主应用组件（含油糕品牌注释）
│   ├── main.tsx            # 应用入口
│   ├── components/         # 所有UI组件
│   │   ├── HomePage.tsx    # 首页
│   │   ├── ServicePage.tsx # 服务页面
│   │   └── FeedbackPage.tsx # 反馈页面
│   └── styles/            # 样式文件
├── index.html             # HTML入口（含油糕品牌元信息）
├── package.json           # 项目配置（已品牌化）
└── README.md             # 本文件
```

## 功能特点
- **三页面路由系统**：首页 → 服务页 → 反馈页
- **响应式设计**：自动适配PC和移动设备
- **现代化UI**：深色主题、渐变背景、流畅动画
- **交互组件**：面部认证模态框、服务分析、用户反馈
- **品牌标识**：油糕品牌注释和元数据集成
